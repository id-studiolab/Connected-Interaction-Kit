# Pico 2W Development Board
The central component at the heart of your projects. It runs code, reads and processes sensor data, and controls outputs (lights, motors, speakers, etc.) 

---
The Pico 2W is an RP2350-based development board (dual‑core Cortex‑M0+) with onboard wireless connectivity. It’s a compact, general‑purpose microcontroller board that runs your code, reads sensors, and controls outputs (GPIO, I2C, SPI, UART, PWM, ADC, etc.). GPIO lines are flexible and multiplexed, so check the Pico 2W pinout and plan pin assignments when using multiple peripherals.

You can program the Pico 2W with CircuitPython. For firmware updates, board-specific builds, and setup instructions, refer to the official documentation and community guides:

- Raspberry Pi Pico (RP2350) documentation: [Datasheet](https://pip-assets.raspberrypi.com/categories/1088-raspberry-pi-pico-2-w/documents/RP-008304-DS-1-pico-2-w-datasheet.pdf)
- CircuitPython (Pico W) support and install instructions: [Circuit Python Website](https://circuitpython.org/board/raspberry_pi_pico2_w/)
